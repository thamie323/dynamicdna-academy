# DNA Academy Design Analysis

## Key Design Elements from Reference Site

### Color Scheme
- **Primary Orange**: Vibrant orange (#E67E22 or similar) - used for hero background, accents, CTAs
- **Navy Blue**: Dark navy (#1A2332 or similar) - used for text, secondary elements
- **White/Light**: Clean white backgrounds for content sections
- **Text**: Dark navy for headings, gray for body text

### Typography
- **Headings**: Bold, modern sans-serif (appears to be system fonts)
- **Body**: Clean, readable sans-serif
- **Style**: Mix of regular and bold weights, large heading sizes

### Layout Structure
1. **Header**: 
   - Logo on left (DNAcademy text-based)
   - Horizontal navigation menu (Home, About, Programs, Corporate, News)
   - "Contact Us" button on right (orange)
   - White/light background with subtle shadow

2. **Hero Section**:
   - Full-width orange background
   - Left: Large white text headline + description + two CTA buttons
   - Right: Image with rounded corners
   - Wave divider at bottom (curved SVG)
   - Tagline above main headline

3. **Programs Section**:
   - White background
   - "Our Programs" heading with orange accent
   - Three cards in a row with subtle borders
   - Icons at top of each card (orange/navy)
   - "View Details" buttons

4. **Mission Section**:
   - Two-column layout
   - Left: Image with overlay card ("Our Mission" badge)
   - Right: Large heading with orange accent word + description
   - Statistics/metrics below

5. **Corporate Section**:
   - Light background
   - Grid of service items
   - "Get Started" CTA button

6. **News Section**:
   - Four cards in a row
   - Category badges
   - Timestamps
   - "Read More" links

7. **Footer**:
   - Dark navy background
   - Multi-column layout
   - Social media icons

### Key Features to Implement
- Wave/curve dividers between sections
- Card-based layouts with subtle shadows
- Orange accent color for CTAs and highlights
- Rounded corners on images and cards
- Clean, spacious design with good whitespace
- Responsive grid layouts

### Interactive Elements Needed
- "Become a Learner" registration form
- "Become a Client" registration form
- Dynamic form handling (not static)
- Form validation
- Success/error states

### Assets to Create Locally
- No Google Fonts CDN - use system fonts or self-hosted
- Local SVG wave dividers
- Local icons (using lucide-react is fine)
- All images stored locally
